📘 Introduzione

Questa guida descrive le procedure per accedere a FTC HUB, effettuare il primo login e gestire il proprio profilo utente, incluso il cambio password.

🌐 Indirizzo applicazione: https://hub.nexturelab.com
🔑 1. Accesso al sistema

Questa sezione descrive la procedura per accedere a FTC HUB.

  1. Aprire il browser (Chrome, Edge, Safari, Firefox) e navigare all'indirizzo https://hub.nexturelab.com
  2. Nella schermata di login, inserire il proprio Username nel campo apposito (vedi tabella sotto per il formato)
  3. Inserire la Password ricevuta
  4. Cliccare il pulsante Accedi
  5. Se le credenziali sono corrette, si verrà reindirizzati alla Dashboard principale

⚠️ In caso di errore: Se appare il messaggio "Credenziali non valide", verificare di aver inserito correttamente username e password. Dopo più tentativi falliti, contattare l'amministratore di sistema.
✅ Sessione persistente: La sessione rimane attiva: non è necessario effettuare il login ad ogni visita.
🆕 2. Primo accesso

Cosa fare al primo accesso al sistema.

  1. Al primo accesso, utilizzare la password temporanea ricevuta via email o comunicata dal proprio responsabile
  2. Dopo aver effettuato il login, cambiare immediatamente la password dalla sezione Profilo (vedi Sezione 3)
  3. Scegliere una password sicura e personale
🔒 Attenzione: Si consiglia di utilizzare una password diversa da quella utilizzata per l'accesso ad altri servizi aziendali.
👤 3. Gestione Profilo

Come accedere e modificare i propri dati personali e la password.

3.1 — Accedere al Profilo

  1. Nella barra laterale sinistra (sidebar), individuare il proprio nome utente visualizzato in basso
  2. Cliccare sul proprio nome per aprire la pagina Profilo
📸 Screenshot: sidebar con il nome utente evidenziato in basso a sinistra

3.2 — Modifica Dati Personali

  1. Nella sezione "Dati personali" è possibile modificare: Username, Email, Telefono
  2. Apportare le modifiche desiderate nei campi editabili
  3. Cliccare il pulsante Salva modifiche per confermare
  4. Le modifiche sono immediate e attive da subito
📸 Screenshot: sezione "Dati personali" con campi Username, Email, Telefono e pulsante Salva modifiche

3.3 — Cambio Password

  1. Nella sezione "Cambia password", inserire la Password attuale
  2. Inserire la Nuova password scelta
  3. Reinserire la nuova password nel campo Conferma password
  4. Cliccare il pulsante Aggiorna password
  5. La nuova password è attiva immediatamente: dal prossimo login usare la nuova password
📸 Screenshot: sezione "Cambia password" con campi password attuale, nuova password, conferma e pulsante "Aggiorna password"
💡 Suggerimento: Per una password sicura, utilizzare almeno 8 caratteri, includendo lettere maiuscole, minuscole, numeri e un carattere speciale (es. ! @ #).
